FAQ
Hva er den beste måten å lære seg å kode?:
Den beste måten å lære seg å kode er å lære av andre og jobbe med det på fritiden. Det kan også være god
øvelse og lære vekk det du kan til andre og.
Hvilke program er best å bruke får å kode?:
Den beste i min mening er Visual Studio Code, men Notepad++ kan også være en mulighet.
Hva slags utdanning kan du få med IT-linjen?
Hvis du utdanner deg vi IT-linjen kan du utdanne deg til å bli utvikler eller drift. Du vil også lære om
brukerstøtte som du mest sannsynelig vil møte i starten av lærling tiden din.
Hvilke kode språk er det mest brukfor?:
HTML, CSS og JavaScript er veldig ofte brukte språk, men Python og C#/C++ er også en viktig del får scripting.
Hvordan snakke med jenter?:
Du gjør ikke. Dette er IT-linjen, hva trodde du?
